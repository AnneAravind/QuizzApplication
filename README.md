# Quiz Application

A RESTful Quiz Application built using Spring Boot and Spring Data JPA. The application provides APIs to manage quiz questions and demonstrates backend development, database integration, and CRUD operations.

## Features
- Create quiz questions
- View all quiz questions
- Update existing questions
- Delete quiz questions
- Store and retrieve quiz data from a database
- Expose REST APIs for quiz management

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- REST API

## Concepts Used
- CRUD Operations
- RESTful API Development
- Backend Design
- Database Integration
- Exception Handling

## How to Run
1. Clone this repository.
2. Create a MySQL database.
3. Update database configuration in `application.properties`.
4. Run the Spring Boot application.
5. Test the APIs using Postman or a web browser.
